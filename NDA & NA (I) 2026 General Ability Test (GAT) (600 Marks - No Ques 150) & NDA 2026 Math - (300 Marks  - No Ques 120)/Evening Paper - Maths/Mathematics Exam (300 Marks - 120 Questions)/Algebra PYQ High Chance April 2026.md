## Very Important Notes and Formulas

### Sets and Operations
- Set: collection of distinct objects. Example: A = {1,2,3}.
- Union A ∪ B: all elements in A or B.
- Intersection A ∩ B: elements in both A and B.
- Difference A \ B: elements in A that are not in B.
- Complement A': elements not in A in the universal set.
- Empty set ∅ has no elements.
- Cardinality |A| is the number of elements in A.
- Venn diagrams show relationships among sets.
- De Morgan's laws: (A ∪ B)' = A' ∩ B', (A ∩ B)' = A' ∪ B'.
- Cartesian product A × B is the set of ordered pairs (a,b) with a ∈ A and b ∈ B.

### Relation and Equivalence Relation
- Relation from A to B is any subset of A × B.
- If (a,b) is in relation R, then aRb.
- Reflexive: every a ∈ A satisfies aRa.
- Symmetric: if aRb then bRa.
- Transitive: if aRb and bRc then aRc.
- Equivalence relation: reflexive, symmetric, and transitive.
- Equivalence classes partition the set.

### Real Numbers on a Line
- Real numbers can be represented on the number line.
- Each point corresponds to a unique real number and vice versa.
- Rational numbers are fractions; irrational numbers are non-repeating, non-terminating decimals.
- Absolute value |x| measures distance from 0.
- Interval notation: [a,b], (a,b), [a,b), (a,b].

### Complex Numbers
- Complex number z = a + bi, where a is real part and b is imaginary part.
- i^2 = -1.
- Modulus |z| = √(a^2 + b^2).
- Argument arg(z) = tan^{-1}(b/a) with quadrant adjustments.
- Conjugate of z is a - bi.
- Polar form: z = r(cos θ + i sin θ).
- Euler form: z = r e^{iθ}.
- Cube roots of unity: 1, ω, ω^2 with ω = -1/2 + i√3/2 and ω^3 = 1, ω ≠ 1.
- Properties: 1 + ω + ω^2 = 0, ω^2 = ω̄.

### Binary Numbers
- Binary system is base 2. Digits are 0 and 1.
- Decimal to binary: divide by 2 repeatedly and read remainders backward.
- Binary to decimal: sum digits × 2^position.
- Example: decimal 13 = binary 1101.
- Binary arithmetic uses base-2 addition and subtraction.

### Progressions
- AP: a, a+d, a+2d, ..., a+(n-1)d.
  - nth term: a_n = a + (n-1)d.
  - Sum: S_n = n/2 [2a + (n-1)d].
- GP: a, ar, ar^2, ..., ar^{n-1}.
  - nth term: a_n = ar^{n-1}.
  - Sum (finite): S_n = a(1 - r^n)/(1 - r) if r ≠ 1.
  - Infinite sum: S = a/(1 - r) if |r| < 1.
- HP: terms are reciprocals of an AP.
  - If terms are 1/a, 1/(a+d), ..., then HP sum uses harmonic mean relations.

### Quadratic Equations
- Standard form: ax^2 + bx + c = 0 with real coefficients.
- Discriminant D = b^2 - 4ac.
  - D > 0: two distinct real roots.
  - D = 0: one repeated real root.
  - D < 0: complex roots.
- Roots: x = [-b ± √D]/(2a).
- Sum of roots = -b/a.
- Product of roots = c/a.
- Graph is a parabola opening up if a > 0, down if a < 0.

### Linear Inequations of Two Variables by Graphs
- Inequation ax + by < c represents a half-plane.
- Graph the boundary line ax + by = c, then test a point.
- For ≤ or ≥, boundary line is included.
- Intersection of two half-planes gives the solution region.

### Permutations and Combinations
- Permutation: order matters. P(n,r) = n!/(n-r)!.
- Combination: order does not matter. C(n,r) = n!/[r!(n-r)!].
- Relationship: P(n,r) = r! C(n,r).
- Circular permutation of n distinct objects = (n-1)!.
- Repetition allowed: n^r.

### Binomial Theorem
- (a + b)^n = Σ_{k=0}^n C(n,k) a^{n-k} b^k.
- Binomial coefficient: C(n,k) = n!/[k!(n-k)!].
- Middle term for odd n is single term, for even n there are two middle terms.
- Useful for expansion, coefficients, and approximation.

### Logarithms
- log_b a = c means b^c = a.
- log_b (xy) = log_b x + log_b y.
- log_b (x/y) = log_b x - log_b y.
- log_b (x^n) = n log_b x.
- log_b b = 1, log_b 1 = 0.
- Change of base: log_b a = log_c a / log_c b.
- Applications: solving exponential equations, growth/decay, pH, intensity scales.
