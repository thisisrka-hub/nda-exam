# NDA 2026 Mathematics: Trigonometry (15-18%)

## Important Notes and Formulas

### Identities
- **Basic Ratios**: sinθ = opposite/hypotenuse, cosθ = adjacent/hypotenuse, tanθ = opposite/adjacent
- **Reciprocal Identities**: cscθ = 1/sinθ, secθ = 1/cosθ, cotθ = 1/tanθ
- **Pythagorean Identities**: sin²θ + cos²θ = 1, 1 + tan²θ = sec²θ, 1 + cot²θ = csc²θ
- **Sum and Difference**: sin(A±B) = sinA cosB ± cosA sinB, cos(A±B) = cosA cosB ∓ sinA sinB, tan(A±B) = (tanA ± tanB)/(1 ∓ tanA tanB)
- **Double Angle**: sin2θ = 2sinθ cosθ, cos2θ = cos²θ - sin²θ = 2cos²θ - 1 = 1 - 2sin²θ, tan2θ = 2tanθ/(1 - tan²θ)
- **Triple Angle**: sin3θ = 3sinθ - 4sin³θ, cos3θ = 4cos³θ - 3cosθ
- **Product Identities**: sinA sinB = [cos(A-B) - cos(A+B)]/2, cosA cosB = [cos(A-B) + cos(A+B)]/2, sinA cosB = [sin(A+B) + sin(A-B)]/2

### Inverse Functions
- **Domain and Range**: sin⁻¹x ∈ [-π/2, π/2], cos⁻¹x ∈ [0, π], tan⁻¹x ∈ (-π/2, π/2)
- **Principal Values**: sin⁻¹(sinθ) = θ for θ ∈ [-π/2, π/2], etc.
- **Identities**: sin⁻¹x + cos⁻¹x = π/2, tan⁻¹x + cot⁻¹x = π/2
- **Sum and Difference**: sin⁻¹x + sin⁻¹y = sin⁻¹(x√(1-y²) + y√(1-x²)) for x,y ≥0, etc.
- **Properties**: sin⁻¹(-x) = -sin⁻¹x, cos⁻¹(-x) = π - cos⁻¹x, tan⁻¹(-x) = -tan⁻¹x

### Heights & Distances
- **Angle of Elevation/Depression**: tanθ = height/distance
- **Right Triangle Applications**: Use sin, cos, tan for sides and angles.
- **Problems**: Involve trees, buildings, towers; use trigonometric ratios.
- **Formulas**: For height h, distance d, angle θ: h = d tanθ, d = h / tanθ

### Triangle Properties
- **Sine Rule**: a/sinA = b/sinB = c/sinC = 2R
- **Cosine Rule**: a² = b² + c² - 2bc cosA, etc.
- **Area**: (1/2)ab sinC, (1/2)bc sinA, (1/2)ca sinB
- **Projection Rule**: a = b cosC + c cosB
- **Napier's Analogy**: tan((A-B)/2) = (a-b)/(a+b) cot(C/2)
- **Circumradius R = abc/(4K)**, Inradius r = K/s, where s = (a+b+c)/2, K = area
- **Angle Bisector**: In triangle, bisector divides opposite side in ratio of adjacent sides.
